# Brain Games

Набор консольных игр для развития математических навыков.

## Игры

1. **Even** — проверка числа на чётность
2. **Calculator** — вычисление арифметических выражений
3. **GCD** — поиск наибольшего общего делителя
4. **Progression** — поиск пропущенного числа в прогрессии
5. **Prime** — проверка числа на простоту
